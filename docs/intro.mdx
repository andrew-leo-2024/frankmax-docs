# Frankmax CMS Documentation

Welcome to the Frankmax CMS documentation. This site provides comprehensive guides for content editors, developers, and system administrators.

## What is Frankmax CMS?

Frankmax CMS is a governance-first content management system built on [Payload CMS](https://payloadcms.com/) and [Next.js](https://nextjs.org/). It powers **250+ country-specific websites** for Frankmax Digital, a global digital consulting corporation offering **48 service offerings** across **5 sub-brands**.

### Key Features

- **Block-based page builder** — 20 content blocks (Hero Banner, Bento Grid, Carousel, Stats, and more)
- **Multi-jurisdiction targeting** — Content can target specific countries or be globally visible
- **Sovereign theming** — Each jurisdiction gets a unique color palette derived from its flag colors
- **Governance workflows** — Draft → Review → Ratified → Published lifecycle with audit logging
- **Campaign management** — Timeline/Gantt view, KPI targets, and multi-content linking
- **Live preview** — Real-time preview with mobile, tablet, and desktop breakpoints
- **5 custom admin dashboards** — Sites, Site Detail, Campaigns, Content Orchestrator, Analytics
