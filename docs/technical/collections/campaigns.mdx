# Campaigns Collection

The Campaigns collection manages marketing campaigns that aggregate content across multiple collections and target specific jurisdictions.

## Configuration

| Property | Value |
|----------|-------|
| **Slug** | `campaigns` |
| **Admin Group** | Content |
| **Versions/Drafts** | `true` |

### Status Options

| Value | Label | Description |
|-------|-------|-------------|
| `draft` | Draft | Campaign is being planned |
| `scheduled` | Scheduled | Campaign is scheduled for future activation |
| `active` | Active | Campaign is currently running |
| `paused` | Paused | Campaign is temporarily suspended |
| `completed` | Completed | Campaign has ended |

### Related Content (Polymorphic)

The `relatedContent` field is an array where each entry contains a polymorphic relationship that can reference any of the following collections:

- `pages`
- `services`
- `news-articles`
- `products`

This allows a single campaign to aggregate different types of content.

```typescript
{
  name: 'relatedContent',
  type: 'array',
  fields: [
    {
      name: 'content',
      type: 'relationship',
      relationTo: ['pages', 'services', 'news-articles', 'products'],
    },
  ],
}
```

### KPI Targets

The `kpiTargets` field is an array of metric tracking entries:

| Sub-field | Type | Description |
|-----------|------|-------------|
| `metric` | text | Name of the KPI (e.g., "Page Views", "Conversions") |
| `target` | number | Target value for the metric |
| `actual` | number | Actual achieved value |

## Admin Views

Campaigns have a dedicated admin dashboard at `/admin/campaigns-dashboard` that provides:

- Timeline visualization with positioned bars based on start/end dates
- Status color coding
- Budget and content count display

See [Campaigns Dashboard](../admin-views/campaigns-dashboard) for details.

| Method | Endpoint | Description |
|--------|----------|-------------|

| GET | `/api/campaigns/:id` | Get a specific campaign |
| PATCH | `/api/campaigns/:id` | Update a campaign |
| DELETE | `/api/campaigns/:id` | Delete a campaign |
